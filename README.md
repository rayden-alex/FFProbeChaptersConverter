This program is designed to convert chapter data generated by **FFProbe** (from the **FFMpeg** package) into **CUE** or **CSV** format.

    Program usage: FFProbeChaptersConverter [-h] [-i <FILE>] [-o <FILE>]
    
     Convert FFProbe chaptersList from JSON to CUE or CSV format.
    
            Options                          Description
     -i, --in-file <FILE>      Input file. Default is '-' for STDIN.
     -o, --out-file <FILE>     Output file. Default is '-' for STDOUT.
     -f, --format <CUE, CSV>   Output format. Default is CUE.
     -h, --help                Show usage help.

Example FFProbe call:

    ffprobe -i %in_file% -of json -show_format -show_streams -show_chapters

This produces JSON like this:

    "chapters": [
            {
                "id": 0,
                "time_base": "1/1000",
                "start": 0,
                "start_time": "0.000000",
                "end": 280000,
                "end_time": "280.000000",
                "tags": {
                    "title": "Lynnic, ItsArius & Dinia - Maze Of Memories"
                }
            },
            {
                "id": 1,
                "time_base": "1/1000",
                "start": 280000,
                "start_time": "280.000000",
                "end": 455000,
                "end_time": "455.000000",
                "tags": {
                    "title": "Lynnic, ItsArius & Dinia - Alone"
                }
            }

Additionally, data about the file format and its media streams is generated.

The **FFProbeChaptersConverter** program converts this data into a **CUE** format like this:

    REM GENRE Music
    REM DATE 2025
    PERFORMER "Lynnic, ItsArius"
    TITLE "Melodic House Living Room Session with Live Vocals - Lynnic & ItsArius"
    FILE "Melodic House Living Room Session with Live Vocals - Lynnic & ItsArius [CAGzxEDwAts].opus" WAVE
      TRACK 01 AUDIO
        TITLE "Lynnic, ItsArius & Dinia - Maze Of Memories"
        INDEX 01 00:00:00
      TRACK 02 AUDIO
        TITLE "Lynnic, ItsArius & Dinia - Alone"
        INDEX 01 04:40:00

This allows you to further split the source file into several using **Paracoder** (https://github.com/rayden-alex/paracoder)

The **FFProbeChaptersConverter** program can also convert chapters data into a **CSV** format like this:

    0.000,280.000,1,"Lynnic, ItsArius & Dinia - Maze Of Memories"
    280.000,455.000,2,"Lynnic, ItsArius & Dinia - Alone"
    455.000,605.000,3,ItsArius & Lynnic - Cornfield Chase
    605.000,823.000,4,Lynnic & ItsArius feat. Lilly - Self Control
    823.000,996.000,5,Lynnic - Rise (feat. Lilly)
    996.000,1202.000,6,"Lynnic, ItsArius & Mocean - Keep Breathing"
    1202.000,1470.560,7,Lynnic & Mocean - Together

For easy use, **FFProbeChaptersConverter.cmd** has been created.
You can place its shortcut in the *%USERPROFILE%\SendTo* folder for easy use in Windows Explorer.
